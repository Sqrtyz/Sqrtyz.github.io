K-D Tree 学习笔记

2020-03-23 1 min read

K-D Tree 是一种类似平衡树的可以存储多维数据的数据结构，在 kkk 比较小时可以体现出优异的性能。

引入

考虑这样一个问题：

  • 支持在平面上插入一个点，询问离某个点最近的点的距离。

  • 强制在线，n,m≤105n,m \leq 10^5n,m≤105。

K-D Tree 流程

基本思想

将平面上的所有坐标用一个类似平衡树的数据结构维护，每个节点存储坐标的有关信息。
