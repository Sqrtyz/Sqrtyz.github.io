[SCOI 2016] 背单词

首先这是一道语文题（233）

翻译题目后，我们可以知道有如下的题意：

  • 给你 nnn 个字符串，不同的排列有不同的代价，代价按照如下方式计算（字符串 sss 的位置为 xxx）：
    1. 排在 sss 后面的字符串有 sss 的后缀，则代价为 n2n^2n2；
    2. 排在 sss 前面的字符串有 sss 的后缀，且没有排在 sss 后面的 sss 的后缀，则代价为 x−yx-yx−y（yyy 为最后一个与 sss 不相等的后缀的位置）；
    3. sss 没有后缀，则代价为 xxx。
  • 求最小代价和。

考虑建立 trie 树。然而 trie 树维护的是前缀，此题要求后缀。不需要很多麻烦的操作，因为此题答案只和后缀有关，把串反过来就好了。

然后在 trie 树上建一个图，具体地，trie 树上如果一个结尾点位于结尾点的子树，从深度小的向深度大的连边。

然后依照 dfs 序跑一次贪心就好了。

[SCOI 2016] 萌萌哒

首先 O(n2)O(n^2)O(n2) 的做法比较好想，搞一个并查集就好了。具体地，对于 l1∼r1l1 \sim r1l1∼r1 中每一个位置向 l2∼r2l2 \sim r2l2∼r2 对应的位置建立附属关系，最后 O(n)O(n)O(n) 查询得解。

然而只能获得部分分，因此考虑优化。

注意到只有一次询问，我们能否将修改和查询都降至 log⁡\loglog 级别呢？

这时，倍增的思想就呼之欲出了。我们将并查集分为 log⁡n\log nlogn 层，以位置 iii 起始的第 jjj 层并查集维护了 i∼i+2j−1i \sim i + 2^j - 1i∼i+2j−1 的信息。在修改的时候，把 l∼rl \sim rl∼r 分为若干个 222 的次幂的块，在对应的并查集层级维护信息。

查询也是 log⁡\loglog 级别。我们只需要依次分裂并查集，对于以位置 iii 起始的第 jjj 层并查集，我们可以将其分裂为以位置 iii 起始的第 j−1j-1j−1 层并查集和以位置 i+2(j−1)i+2^(j-1)i+2(j−1) 起始的第 j−1j-1j−1 层并查集，直至分裂到只剩第 000 层，这时直接查询即可。

假设你查询的答案为 xxx，注意答案并不是 10x10^x10x，而是 9×10x−19 \times 10^{x-1}9×10x−1。

Luogu P4735 最大异或和

可持久化 01-trie。

考前只学过主席树，没学过这玩意儿。但是类比上来还是很容易的，但是由于我太菜，在其他地方又栽了个跟头……

回到正题。考虑如果是询问 a1⊕a2⊕...⊕ap⊕xa_1 \oplus a_2 \oplus ... \oplus a_p \oplus xa1​⊕a2​⊕...⊕ap​⊕x 的最大值 ，令 bi=a1⊕a2⊕...⊕aib_i = a_1 \oplus a_2 \oplus ... \oplus a_ibi​=a1​⊕a2​⊕...⊕ai​，则相当于在 l∼rl \sim rl∼r 中找到 ppp 使得 bp⊕xb_p \oplus xbp​⊕x 最大，这是可持久化 Trie 树的裸题。

但是现在是询问 ap⊕ap+1⊕...⊕an⊕xa_p \oplus a_{p+1} \oplus ... \oplus a_n \oplus xap​⊕ap+1​⊕...⊕an​⊕x 。 考虑对 a1⊕a2⊕...⊕ap−1⊕xa_1 \oplus a_2 \oplus ... \oplus a_{p-1} \oplus xa1​⊕a2​⊕...⊕ap−1​⊕x 异或上 bn=a1⊕...⊕anb_n = a_1 \oplus ... \oplus a_nbn​=a1​⊕...⊕an​ 则为所求式子 。

于是就很简单了，维护前缀异或和和前缀异或和的可持久化 01-trie，查询时直接令 q=bn⊕xq=b_n \oplus xq=bn​⊕x，让 qqq 在 l−1l-1l−1 和 r−1r-1r−1 之间的 01-trie 跑贪心就可以了。
